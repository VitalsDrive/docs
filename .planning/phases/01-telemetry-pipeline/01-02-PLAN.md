<!-- FMC003 IO ID hex reference for buffer writes — uint16 BE -->

| Decimal | Hex    | Field             | Scaling (sim writes)              | IO type |
|---------|--------|-------------------|-----------------------------------|---------|
| 7040    | 0x1B80 | Coolant temp      | uint16 BE = temp × 10             | 2       |
| 7044    | 0x1B84 | Engine RPM        | uint32 BE = rpm                   | 3       |
| 7045    | 0x1B85 | Vehicle speed     | uint16 BE = speed km/h            | 2       |
| 7059    | 0x1B93 | Control voltage   | uint16 BE = voltage × 1000 (mV)   | 2       |

Buffer layout per IO element: [ID hi, ID lo, type byte, ...value bytes]
- uint16 value: 4 bytes total → Buffer.alloc(4); buf.writeUInt16BE(IO_ID, 0); buf[2] = type; buf.writeUInt16BE(value, 3).
- uint32 value: 6 bytes total → Buffer.alloc(6); buf.writeUInt16BE(IO_ID, 0); buf[2] = type; buf.writeUInt32BE(value, 3).
</interfaces>
