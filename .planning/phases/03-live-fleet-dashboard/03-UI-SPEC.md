## Spacing Scale

All values from `--vd-space-*` tokens. Multiples of 4px base grid.

| Token | CSS Var | Value | Usage |
|-------|---------|-------|-------|
| xs | `--vd-space-1` | 4px | Icon gaps, dot indicators |
| sm | `--vd-space-2` | 8px | Compact inline spacing, card internal gaps |
| md | `--vd-space-4` | 16px | Default element spacing, form field gaps |
| lg | `--vd-space-6` | 24px | Section padding, card body padding, grid gap |
| xl | `--vd-space-8` | 32px | Layout gaps between sections |
| 2xl | `--vd-space-12` | 48px | Loading/empty state vertical padding |
| 3xl | `--vd-space-16` | 64px | Page-level top spacing, header height |

Exceptions:
- Connection pill: 5px vertical / 12px horizontal padding — 12px is `--vd-space-3` (valid token). 5px is not a multiple of 4 and is an existing implementation value on `VdConnectionPillComponent` — do not change.
- Map full-bleed: 0 padding on `FleetMapComponent` host element (Leaflet needs full viewport)
- Touch targets for map marker click zones: minimum 44px × 44px

## Color

All values from `--vd-*` tokens. 60/30/10 split across dark warm-umber palette.

| Role | Hex | CSS Var | Usage |
|------|-----|---------|-------|
| Dominant (60%) | #0e0a07 | `--vd-bg-base` | Page background, map overlay underlay |
| Secondary (30%) | #1a130d | `--vd-bg-surface` | Vehicle cards, sidebar, header, detail panel |
| Accent (10%) | #f59e0b | `--vd-brand` | Primary CTA buttons, active nav item indicator |
| Elevated surface | #241a12 | `--vd-bg-elevated` | Hovered cards, map info overlay, modal backgrounds |
| Destructive | #ef4444 | `--vd-critical` | DTC alert indicators, offline state, error banners only |

Accent reserved for:
- `[+ Add Vehicle]` CTA button on empty state
- Active/selected sidebar nav item left-border indicator
- Connection pill "reconnecting" state uses `--vd-warning` (#eab308) — not accent

Vehicle state color semantics (non-negotiable, matches existing `ConnectionPillComponent` and `FleetMapComponent`):

| State | Color | Hex | CSS Var |
|-------|-------|-----|---------|
| running | Lime | #84cc16 | `--vd-healthy` |
| alert | Yellow | #eab308 | `--vd-warning` |
| offline | Muted umber | #5a4530 | `--vd-offline` |
| stale (new) | Grey-dimmed | #5a4530 at 50% opacity | `--vd-offline` + `opacity: 0.5` |
| critical/DTC | Red | #ef4444 | `--vd-critical` |

Map marker state colors follow the same palette. Stale marker: grey DivIcon with `opacity: 0.5` applied to the entire marker element.

## Interaction Contracts

### Vehicle Card Click (`/dashboard`)
- Entire card is clickable. Cursor: `pointer`.
- Click → navigate to `/vehicle/:id` (decision D-03).
- Hover: `border-color` transitions to `--vd-border-strong` (#503624) at 220ms `--vd-ease-out`. Box-shadow elevates to `--vd-shadow-elevated`.
- No secondary action other than navigation.

### Map Marker Click (`/map`)
- Click → `selectedVehicleId` signal set → `VehicleDetailPanelComponent` slides in from right.
- Slide-in animation: translate X from +100% to 0, 220ms `--vd-ease-out`.
- Stale markers: still clickable. Panel shows "Last seen X min ago" prominently in the header area.
- Touch target: minimum 44px × 44px DivIcon clickable area.

### Connection Status Transitions
- `connected → disconnected`: toast fires immediately. Copy: "Connection lost — reconnecting…" Toast duration: persistent until state changes.
- `disconnected → connected`: toast fires. Copy: "Live data restored." Toast duration: 3000ms then auto-dismiss.
- `reconnecting` state: pill pulses (existing animation). No additional toast for intermediate reconnecting state.

### Empty States
- `/dashboard` no vehicles: render `LoadingSpinnerComponent` while resource is pending. Once resolved with empty array: show empty state block (see Copywriting).
- `/map` no GPS data: map tiles render at default center `[32.0853, 34.7818]`, zoom 12. Centered overlay div appears above map canvas with semi-transparent background `rgba(14, 10, 7, 0.75)`, border-radius `--vd-radius-lg`.

### Resource Loading States
- While `resource()` status is `'loading'`: show `LoadingSpinnerComponent` centered in content area. No skeleton — spinner only.
- While `resource()` status is `'error'`: show error banner (matches `.error-banner` pattern from fleet-management: red border + background + icon).

### Staleness (Decision D-15 to D-18)
- Threshold: 15 minutes. Evaluated on every marker render pass.
- Stale state takes priority over all other states in `getVehicleState()`.
- No pulsing animation on stale markers (static grey DivIcon).
- Tooltip text format: "Last seen {N} min ago" — always show minutes, never hours, for values under 24h.

```
┌─────────────────────────────────────────────────┐
│ Shell Header (64px)  [brand] [nav] [conn-pill]  │
├──────────┬──────────────────────────────────────┤
│ Sidebar  │  Page padding: 24px all sides        │
│ (240px)  │  Page heading: "Fleet Dashboard"     │
│          │  ─────────────────────────────────── │
│          │  Vehicle grid: auto-fill 280px cols  │
│          │  Grid gap: 24px                      │
│          │  [Card] [Card] [Card]                │
│          │  [Card] [Card] [Card]                │

```
- Grid: `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))`
- Grid gap: `--vd-space-6` (24px)
- Card accent bar: 4px left edge, color reflects vehicle state

### `/map` route
```
┌─────────────────────────────────────────────────┐
│ Shell Header (64px)  [brand] [nav] [conn-pill]  │
├──────────┬──────────────────────────────────────┤
│ Sidebar  │  Map canvas: 100% width, 100% height │
│ (240px)  │  (full-bleed, zero padding)          │
│          │  ┌────────────────────────────────┐  │
│          │  │  Leaflet tiles                 │  │
│          │  │  DivIcon markers               │  │
│          │  └────────────────────────────────┘  │
│          │  VehicleDetailPanel: right slide-in  │
│          │  Width: 360px, full height           │
└──────────┴──────────────────────────────────────┘
```

- Map host: `height: calc(100vh - 64px)` (subtract header height)
- Detail panel: `position: absolute; right: 0; top: 0; bottom: 0; width: 360px`
- Detail panel background: `--vd-bg-surface` (#1a130d), border-left: 1px `--vd-border`
